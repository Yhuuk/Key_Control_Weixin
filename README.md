# 说明文档
| 文件/目录                         | 是否提交 Git | 说明                |
| ----------------------------- | -------: | ----------------- |
| `app.js`                      |       提交 | 小程序入口 JS          |
| `app.json`                    |       提交 | 全局页面、窗口配置         |
| `app.wxss`                    |       提交 | 全局样式              |
| `pages/`                      |       提交 | 页面源码              |
| `components/`                 |       提交 | 组件源码              |
| `sitemap.json`                |       提交 | 小程序索引配置           |
| `.eslintrc.js`                |       提交 | 代码规范配置            |
| `project.config.json`         |     一般提交 | 项目公共配置            |
| `project.private.config.json` |   **忽略** | 个人微信开发者工具配置，不建议提交 |
